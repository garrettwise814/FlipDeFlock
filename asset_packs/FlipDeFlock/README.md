# FlipDeFlock asset pack — "Scanner HUD"

Themed for FlipDeFlock:
- **Desktop animation** — a glitch/scanner-HUD: a sweeping radar line reveals
  detected "blips", locks a reticle on them, with a live spectrum bar and an
  occasional scanline glitch. 128×64, 1-bit, 16 frames.
- **Lock screen** (`Icons/Interface/Lockscreen.png`) — a matching "LOCKED"
  scanner HUD with a padlock, shown when the Flipper is locked.

(The power-on boot splash is firmware-level and not overridable via an SD asset
pack; the desktop animation is the standby/idle screen.)

This folder is the **editable source** (PNG frames + `meta.txt` + `manifest.txt`,
generated by `make_pack.py`). The Flipper loads a **packed** (`.bm`) version.

## Install on the Flipper (Momentum)

1. Get the **packed** pack:
   - Download `flipdeflock-assetpack.zip` from the [release](../../releases/latest), **or**
   - run Momentum's packer yourself: drop this `FlipDeFlock/` folder next to
     `scripts/asset_packer.py` in the Momentum firmware tree and run it — it emits
     a packed `asset_packs/FlipDeFlock/`.
2. Copy the packed **`FlipDeFlock`** folder to your SD card under **`/asset_packs/`**
   (so you have `SD/asset_packs/FlipDeFlock/Anims/...`).
3. On the Flipper: **Settings → Desktop → Asset Pack** → select **FlipDeFlock**
   (Momentum). The new animation appears on the desktop idle screen.

## Edit / regenerate

```sh
pip install pillow
python make_pack.py        # regenerates Anims/.../frame_*.png + meta.txt + manifest.txt
# then re-pack with Momentum's scripts/asset_packer.py to get the .bm version
```

Tweak the look in `make_pack.py` (frame count, sweep speed, blips, glitch). Source
PNGs are **black-on-white** (black = lit pixel) — the packer inverts + compresses
them to `.bm`.
